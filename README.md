# Sistema de Cadastro Básico em Python

Projeto desenvolvido para praticar lógica de programação e fundamentos da linguagem Python.

## Funcionalidades
- Cadastro de usuário via terminal
- Confirmação de e-mail
- Confirmação de senha
- Controle de fluxo com while e if

## Conceitos estudados
- Variáveis
- input() e print()
- Estruturas condicionais
- Laços de repetição
